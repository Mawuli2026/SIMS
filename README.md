# SIMS

"Good inventory management turns sales data into better business decisions."

## Local setup

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm run dev
```

3. Open the browser at the URL shown by Vite (usually `http://localhost:5173`).
